🖥️ ST7735 TFT Display Driver (C)

A portable, modular, and platform-independent driver for ST7735 TFT displays, designed for embedded systems.

Built with a clean architecture using:

OSAL (Operating System Abstraction Layer)
SPI Bus abstraction
GPIO (Pin) abstraction
Strategy pattern (function pointers)
✨ Features
✅ Platform-independent (STM32, ESP32, bare-metal)
✅ Clean modular architecture
✅ SPI abstraction layer
✅ OSAL-based timing (no HAL dependency)
✅ Configurable via templates
✅ Rotation support (0°, 90°, 180°, 270°)
